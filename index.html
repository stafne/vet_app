DMG – drag & drop installer, recommended for most users  |  ZIP – extract and move to Applications manually

💿 DMG Installation

  1. Download the DMG above.
  2. Double-click the DMG to open it.
  3. Drag Vet into the Applications shortcut.
  4. Eject the DMG.
  5. Open Applications, right-click Vet.app and choose Open.

📦 ZIP Installation

  1. Download the ZIP above.
  2. Double-click the ZIP to extract it.
  3. Drag Vet.app to your Applications folder.
  4. Open Applications, right-click Vet.app and choose Open.
⚠️ macOS Gatekeeper — If you see "Vet.app cannot be opened because it is from an unidentified developer", go to Apple menu → System Settings → Privacy & Security, scroll down to the Vet.app message and click Open Anyway. You only need to do this once after the first install.

🖥️ Terminal Installer (Alternative)

  1. Open Terminal: Cmd ⌘ + Space, type Terminal, press Enter.
  2. Paste the command below and press Enter:
bash <(curl -fsSL https://stafne.github.io/vet_app/install_vet.sh)

The script fetches the latest release and installs Vet.app to your Applications folder automatically.

🔄 Automatic Updates

Vet checks for new releases on every launch and will prompt you to download and install updates automatically. No manual steps are required after the initial installation.
