🐾 Vet

HP4 accelerometry importer and multi-channel waveform viewer for macOS

📦 Download & Install

Download the latest release and run the one-line installer:

bash <(curl -fsSL https://stafne.github.io/vet_app/install_vet.sh)

⬇️ Download Latest ZIP View All Releases

This link always points to the newest release.

🖥️ Terminal Installer (Recommended)

  1. Open Terminal: Cmd ⌘ + Space, type Terminal, press Enter
  2. Copy and paste this command, then press Enter:
bash <(curl -fsSL https://stafne.github.io/vet_app/install_vet.sh)

The script will download the latest release from GitHub and install Vet.app to your Applications folder.

📋 Manual ZIP Installation

  1. Download Vet.zip using the button above.
  2. Double-click the ZIP to extract it.
  3. Drag Vet.app to your Applications folder.
  4. Open Applications, locate Vet.app, right-click it and choose Open.
  5. If you see "Vet.app was blocked because it is not from an identified developer", go to:
    Apple menu → System Settings → Privacy & Security, scroll down to find the message for Vet.app and click Open Anyway.
  6. The app will now launch. You only need to do this once.

🔄 Automatic Updates

Vet checks GitHub for new releases on startup and will prompt you to download and install the update automatically. No manual steps required after the initial installation.
