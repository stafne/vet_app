🖥️ Terminal Installer (Alternative)

  1. Open Terminal: Cmd ⌘ + Space, type Terminal, press Enter
  2. Copy and paste the command below, then press Enter:
bash <(curl -fsSL https://stafne.github.io/vet_app/install_vet.sh)

The script fetches the latest release from GitHub and installs Vet.app to your Applications folder automatically.

📋 DMG Installation

  1. Download the DMG above.
  2. Double-click the DMG to open it, then drag Vet to Applications.
  3. Open your Applications folder, locate Vet.app, right-click it and choose Open.
  4. If you see "Vet.app was blocked…", go to:
    Apple menu → System Settings → Privacy & Security, scroll down to the message for Vet.app and click Open Anyway.
  5. The app will now launch. You only need to do this once.
  6. You can now eject the DMG.

📦 ZIP Installation (Alternative)

  1. Download Vet.zip using the button above.
  2. Double-click the ZIP to extract it.
  3. Drag Vet.app to your Applications folder.
  4. Open Applications, locate Vet.app, right-click and choose Open.
  5. If blocked: Apple menu → System Settings → Privacy & Security → click Open Anyway next to Vet.app.

🔄 Automatic Updates

Vet checks GitHub for new releases on startup and will prompt you to download and install the update automatically. No manual steps required after the initial installation.
